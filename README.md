# Ewan Dashboard

Admin Dashboard for the **Ewan Tourism Platform**, built with Angular.

---

## 🚀 Overview

Ewan is a tourism booking system that allows users to reserve hotels and apartments بسهولة.  
This project represents the **Admin Dashboard**, which provides full control over the platform.

It is designed for:
- System administrators
- Hotel / property owners

---

## 🎯 Features

### 👨‍💼 Admin Features
- Manage clients and users
- Manage drivers
- Control bookings and reservations
- Activate / deactivate users
- Full CRUD operations
- Dashboard analytics and statistics

### 🏨 Hotel Owner Features
- View their properties
- Track bookings
- Monitor performance
- Analyze revenue and statistics
